# Vue 3 + TypeScript + Vite + vant4 + Eslint + Prettierrc


## 一、项目运行步骤

### 拉取代码（以拉取develop分支为例）

git clone /........

## 安装依赖

npm install

### 运行项目

```
npm dev
```

### 打包项目

```
npm run build
```
